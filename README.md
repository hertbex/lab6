Вот как раз это явный пример геометрического определения, где данные - точки в пространстве.

### Что такое эмбеддинг

Эмбеддинг — это плотный числовой вектор фиксированной размерности (например, 384 числа типа `float32`), который языковая модель строит для текста. Ключевое свойство: семантически похожие тексты получают геометрически близкие векторы.

На подумать, а как бы вы обучили такую модель?

```
"кот сидит на подоконнике"   → [0.12, -0.34, 0.87, ...]   ┐
"кошка лежит на окне"        → [0.11, -0.31, 0.85, ...]   ┘ близко
"ракета летит на орбиту"     → [-0.72, 0.55, -0.10, ...]    далеко
```

Расстояние между векторами — мера семантической близости. Чаще всего используют косинусное сходство или евклидово расстояние (для нормализованных векторов они эквивалентны).

В этой лабораторной эмбеддинги считаются с помощью небольшой модели `all-MiniLM-L6-v2`, которая загружается через `sentence-transformers` прямо в процессе:

```python
from embedding import Embeddings

emb = Embeddings()                          # загружает all-MiniLM-L6-v2 при первом вызове
vec = emb.embed("кот сидит на подоконнике") # np.ndarray, shape (384,)
```

P.S. вам никто не мешает заменить потроха модуля embedding.py для того чтобы вызывать большую модель откуда угодно, это пример "из коробки"

## Задача 1: Семантический индекс

### Постановка

Нужно построить структуру данных, которая:
- хранит набор пар `(вектор, payload)`;
- поддерживает вставку нового элемента;
- отвечает на запрос «найти `k` ближайших к данному вектору элементов».

Сложность поиска должна быть **O(k · log N)**, где N — количество элементов в индексе.

### Идея

Дерево. Конкретная реализация — на ваш выбор:

### Интерфейс

Реализуйте совместимый интерфейс:

```python
import numpy as np
from dataclasses import dataclass


@dataclass(frozen=True)
class Record:
    id: int
    text: str


@dataclass(frozen=True)
class SearchResult:
    id: str
    distance: float
    score: float
    path: tuple[str, ...]  # путь по узлам дерева от корня до листа
    payload: Record | None


class YourIndex:
    def insert(
        self,
        vector: np.ndarray,
        item_id: str,
        payload: Record | None = None,
    ) -> None:
        """Добавить вектор в индекс."""
        ...

    def search(
        self,
        query: np.ndarray,
        k: int = 5,
    ) -> list[SearchResult]:
        """Вернуть k ближайших к query элементов."""
        ...
```
